Financial KPI Dashboard (Power BI)

Power BI · DAX · SQL Server

Problem

Leadership needed a reliable, fast KPI view (financial + operational) instead of manual Excel reporting.

Impact
  • Reduced manual reporting effort by standardizing the dashboard refresh and distribution.
  • Improved consistency of KPI definitions across teams through a single semantic layer.
Approach
  • Designed a star-ish reporting dataset (facts + dimensions) using SQL views/stored procedures.
  • Built a KPI layer in Power BI with DAX measures (YTD, variance vs target, rolling trends).
  • Added drill-through pages for branch/product and built bookmarks for guided navigation.
  • Implemented data checks (row counts, null audits, reconciliation totals) before refresh.
